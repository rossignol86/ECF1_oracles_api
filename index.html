Sujet Officiel - ECF Front-End DWWM 2025
Application web et web mobile "Horoscope"

Contexte de l'examen :
Vous travaillez pour une start-up spécialisée dans la diffusion de messages spirituels et horoscopiques. Votre mission consiste à concevoir et développer une application web complète (HTML/CSS/JavaScript) qui se connecte à une API externe et propose une expérience utilisateur fl uide, moderne et responsive.

Conseils :
Votre application doit être simple, élégante, bien codée et utilisable sur mobile. N’oubliez pas les détails qui font la différence : responsive fluide, UI. harmonieuse, code bien commenté et structuré.

Livrables à fournir
À l’issue de l’évaluation, vous devrez transmettre :
> Organisation Scrum Agile + maquette + Kanban Product Backlog + Sprint,
> Lien vers le dépôt GitHub (public),
> Lien vers le site déployé,
> ZIP Projet complet avec les fichiers rangés selon l’arborescence demandée,
> Fichier README.md clair, structuré et professionnel.
Durée de l'épreuve : du 15/07/2025 à 9h00, livraison max le 18/07/2025 à 12h00
